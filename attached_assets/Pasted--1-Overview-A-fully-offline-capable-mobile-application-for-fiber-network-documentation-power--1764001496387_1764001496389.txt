‎1. Overview
‎
‎A fully offline-capable mobile application for fiber network documentation, power mapping, topology management, BT readings, asset tracking, route distance calculation, and automated reporting. Designed for GPON/FTTH/FTTx field engineers, the system allows map-based planning, live field updates, and synchronized storage when the device reconnects online.

‎3. GPON / FTTx Topology Manager
‎

‎FDT / FAT
‎

‎
‎Dome closures
‎
‎Underground closures
‎
‎Aerial closures
‎

‎
‎Location
‎
‎Power Input / Output
‎
‎Loss table
‎
‎Connected segments
‎
‎Status
‎
‎Notes and attachments
